Project Overview

Encryption and decryption program developed using Python, including AES, RSA, and SHA-256 hashing. The program supports key generation, secure storage and retrieval of RSA keys, and interactive user input for encryption and decryption processes.

Features

Symmetric Encryption with AES

  • Allows users to encrypt and decrypt messages using AES in CBC mode.
  • Supports user-defined or program-generated AES keys and IVs.
  • Keys and IVs are managed in a secure base64 format.

Asymmetric Encryption with RSA

  • Generates, saves, and loads RSA public and private keys.
  • Provides encryption with public keys and decryption with private keys.
  • Supports user-specified private keys for decryption.

Hashing with SHA-256

  • Computes and displays secure SHA-256 hashes of user-provided messages.
  • Demonstrates one-way hashing for data integrity verification.

Key Management

  • RSA key pairs are saved to files (private_key.pem and public_key.pem) in PEM format for reuse.
  • AES keys and IVs are generated dynamically or provided by the user.

User Interaction

  • Menu-driven interface with intuitive prompts.
  • Allows users to switch between encryption methods and manage keys seamlessly.

Error Handling

  • Validates user-provided keys, IVs, and encrypted messages.
  • Displays meaningful error messages for decryption or key issues.
